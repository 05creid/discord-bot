?ban <@user> Bans a member from the server
?help Displays all the commands of the server
?mute <@user> <@user> Mutes a user in the server
?unmute <@user> unmutes a user in the server
?afk <@user> sets an AFK message for a user
?warn <@user> Warns a user
?warnings <@user> Displays all the past warnings of a user
?clearwarnings <@user> Clears the warning of a user
?status Changes the status of the bot
?suggest Logs a suggestion to the server
?report <@user> Reports a user
?Invite Generates the invite link from the server
?kick <@user> Kicks a user from the server
?say Broadcasts a message in the channel as the bot
?addrole <@user> (role) adds a role to a user
?softban <@user> Soft bans a user from the server
?removerole <@user> (role) Removes a role from a user
?purge <# of messages> Deletes a number of messages in a channel
?ping Gets the ping of the server and bot
